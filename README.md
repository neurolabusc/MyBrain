# MyBrain

This is a simple repository that allows you to share your brain scan (or other medical image saved as a NIfTI-format image) as an interactive Github Pages web site. You can use dcm2niix to convert most DICOM images to NIfTI. In addition, you can use the online [brainchop](https://brainchop.org/) to brain-extract (scalp strip) brain scans.
